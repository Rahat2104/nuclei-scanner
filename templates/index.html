Docker + Flask + Render + Nuclei

Nuclei Security Scanner

A cloud-based vulnerability scanner that runs Nuclei inside a Docker container, stores scan output in JSONL format, and presents findings through a clean security dashboard.

Start a Scan

Enter an authorized test website URL. For this assignment, use demo sites such as https://demoqa.com/ and https://example.com.

https://demoqa.com/ https://example.com

System Flow

The Flask API receives the target URL, runs Nuclei inside the Docker container, stores JSONL results, and sends the findings back to the dashboard.

Educational use only. Only scan authorized targets, test websites, or your own applications.
